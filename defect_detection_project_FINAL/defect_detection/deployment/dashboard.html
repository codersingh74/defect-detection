Project overview

Manufacturing industries ke liye ek AI-powered Quality Control System jo factory belt par aane wale products mein defects automatically detect karta hai.

Deep Learning models (ResNet-50, YOLOv8) use karke real-time defect detection karta hai with high accuracy aur low latency.

GradCAM explainability se model decision transparent hota hai — inspectors ko dikh sakta hai model kahan dekh raha hai.

Quick start commands
# Install dependencies
pip install -r requirements.txt

# Run EDA
python src/eda.py --data_dir data/raw/

# Train ResNet-50
python src/train.py --model resnet50 --epochs 50

# Launch Streamlit dashboard
streamlit run deployment/app.py

# Launch Gradio demo
python deployment/gradio_demo.py
